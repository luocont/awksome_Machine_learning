# 线性回归最小实例

## 📁 文件说明

- **linear_regression_sample.csv** - 示例数据文件(面积和价格)
- **linear_regression.py** - 线性回归操作脚本
- **linear_regression_README.md** - 本说明文档

## 📊 数据说明

CSV文件包含两列:
- `面积(平方米)` - 房屋面积,作为特征(X)
- `价格(万元)` - 房屋价格,作为目标变量(y)

数据范围: 50-200平方米,价格120-500万元

## 🚀 使用方法

### 1. 安装依赖

```bash
pip install pandas numpy scikit-learn matplotlib
```

### 2. 运行脚本

```bash
python linear_regression.py
```

## 📈 输出内容

运行脚本后会得到:

1. **控制台输出**:
   - 数据预览和统计信息
   - 训练/测试集大小
   - 模型参数(斜率和截距)
   - 回归方程
   - 模型评估指标(MSE、R²)
   - 示例预测结果

2. **可视化图表** (`linear_regression_result.png`):
   - 左图:散点图+回归线
   - 右图:预测值vs实际值对比

## 🔑 核心概念

### 线性回归方程

```
y = wx + b
```

- `y` - 预测值(价格)
- `x` - 特征(面积)
- `w` - 斜率/系数
- `b` - 截距

### 评估指标

- **MSE (均方误差)**: 预测误差的平均值,越小越好
- **R² (决定系数)**: 模型拟合程度,0-1之间,越接近1越好
